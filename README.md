# Web Page Sample

A practice landing page built as part of [The Odin Project](https://www.theodinproject.com/) Foundations curriculum.

## Live Preview

[iceluv0x.github.io/web-page-sample](https://iceluv0x.github.io/web-page-sample)

## About

A fictional landing page for **Black Division** — a corporate security organization from Escape from Tarkov. Built to practice HTML structure and CSS styling.

## What's on the page

- Header with logo and navigation
- Hero section with title, description, and call-to-action button
- Cards section showcasing core operation sectors
- Quote section
- CTA section
- Footer

## Built with

- HTML
- CSS (Flexbox)
- Custom font: Bender
